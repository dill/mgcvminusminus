# mgcvminusminus

<!-- badges: start -->
<!-- badges: end -->

`mgcvminusminus` autogenerates TMB code from `mgcv` model specifications.

## Installation

You can install the released version of `mgcvminusminus` via

``` r
remotes::install_github("dill/mgcvminusminus")
```

## Example

``` r
library(mgcvminusminus)
## basic example code
```
